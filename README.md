Developed an event-based alarm system using Django where users can schedule events and receive desktop alerts at the specified time.

Key Features:
- Create events with title, date, and time
- Automatic alarm trigger when scheduled time is reached
- Audio notification using JavaScript
- View and manage all scheduled events
- Filter events by day, month, and date

Tech Stack:
- Python (Django)
- JavaScript (for alarm/audio handling)
- HTML & CSS

This project helped me understand scheduling logic, time-based event handling, and integration between backend and frontend.
